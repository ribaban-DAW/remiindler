## üè° Usage

First of all, create a `.env` file following the `.env.example`
```
ENV_USER="your_username"
ENV_PASS="your_password"
```

### Linux

1. Create a virtual environment
```
python3 -m venv venv
```

2. Activate the virtual environment
```
source venv/bin/activate
```

3. Run the script
```
python3 main.py
```

### Windows

1. Create a virtual environment
```
python3 -m venv venv
```

2. Activate the virtual environment
```
source venv/bin/activate
```

3. Run the script
```
python3 main.py
```
